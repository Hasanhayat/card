Moniter

A computer monitor is an output device that displays information in pictorial or textual form. A discrete monitor comprises a visual display, support electronics.

CPU

The CPU, also referred to as the “central” or “main” processor, is a complex set of electronic circuitry that runs the machine's operating system and apps.

Graphics card

A graphics processing unit (GPU) is a computer chip that renders graphics and images by performing rapid mathematical calculations.GPUs are also use in gaming.

Keyboard

A computer keyboard is an input device used to enter characters and functions into the computer system by pressing buttons, or keys. It is the primary device used to enter text.

Mouse

A mouse is a small device that a computer user pushes across a desk surface in order to point to a place on a display screen and to select one or more actions.

Motherboard

A motherboard is a circuit board inside general-purpose computing systems, including personal computers, smart televisions, smart monitors, and other similar devices.
